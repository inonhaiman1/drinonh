העלאת האתר ל-GitHub Pages (בקצרה):

1) היכנס/י לריפו: drinon-website
2) לחצו: Add file → Upload files
3) גררו ושחררו את הקובץ index.html (זה הקובץ המצורף כאן) → Commit changes
4) הפעלה/בדיקה של GitHub Pages:
   Settings → Pages → Source: Deploy from a branch
   Branch: main  |  Folder: /(root)
5) אחרי דקה: כנסו לכתובת
   https://inonhaiman1.github.io/drinon-website/
   (אם יש מטמון, רענון קשיח: Ctrl+F5)

טיפים:
- אם כבר קיימים קבצים קודמים, מותר לדרוס את index.html הישן.
- אם מופיעה שגיאה 404, המתינו דקה נוספת ורעננו.
- אפשר להוסיף קישור ישיר לצור קשר: .../index.html#contact
